# EarnOmni Growth Engine

The EarnOmni Growth Engine is a modular, AI-powered SEO and marketing automation
system. It operates independently from the main application and is organized
into seven functional modules:

1. **Research Engine** — keyword, competitor, and market research automation.
2. **On-Page SEO Engine** — site and content audits, technical SEO checks.
3. **Content Engine** — AI-assisted content generation and optimization.
4. **Social Media Design Engine** — automated creative/design generation for social channels.
5. **Ads Intelligence Engine** — ad performance analysis and campaign intelligence.
6. **Dashboard/Approval System** — review and approval workflows for generated output.
7. **Advertiser/Investor Outreach Engine** — automation for advertiser and investor outreach.

## Folder Structure

```
growth-engine/
├── config/              # Configuration files for the growth engine modules
├── migrations/           # Database/schema migrations for growth engine data
├── scripts/
│   ├── research/         # Research Engine scripts
│   ├── seo-audit/        # On-Page SEO Engine scripts
│   ├── content/          # Content Engine scripts
│   ├── social-design/    # Social Media Design Engine scripts
│   ├── ads-intelligence/ # Ads Intelligence Engine scripts
│   └── outreach/         # Advertiser/Investor Outreach Engine scripts
└── README.md
```

This directory is intentionally separate from the main application code.
